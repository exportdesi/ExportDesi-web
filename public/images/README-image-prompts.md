# Export Desi — Image Generation Prompts

**Purpose:** Consistent, professional product imagery for website
**Tool:** Gemini / Midjourney / DALL-E 3
**Style:** Clean, professional, food-grade, export-quality

---

## Usage Notes

- All images should be **high-resolution** (minimum 1920×1080 for hero, 800×600 for product shots)
- **White or neutral backgrounds** for product shots
- **Natural lighting** for farm/processing images
- **Food-safe, clean aesthetics** throughout
- Export as **PNG** for transparency support, **JPG** for photos

## Technical Specifications

### Image Formats
| Use Case | Format | Min Resolution | Max File Size |
|----------|--------|----------------|---------------|
| Hero Images | JPG/PNG | 1920×1080 | 500 KB |
| Product Shots | PNG | 800×600 | 200 KB |
| Process Photos | JPG | 1200×800 | 300 KB |
| Icons/Graphics | SVG | Vector | 50 KB |

### Naming Convention
```
{product}-{subject}-{variant}.{ext}

Examples:
- makhana-hero.png
- makhana-closeup.png
- makhana-grades.png
- turmeric-fingers.png
- dehydrated-onion.png
```

### Color Palette Reference
| Color | Hex | Usage |
|-------|-----|-------|
| Brand Green | #059669 | Primary CTAs, accents |
| Surface | #F9FAFB | Backgrounds |
| Border | #E5E7EB | Dividers, cards |
| Text Muted | #6B7280 | Secondary text |
| Text Primary | #111827 | Headings |

## Prompt Tips

1. **Be specific about lighting:** "studio lighting", "natural daylight", "dramatic side lighting"
2. **Specify background:** "white background", "dark slate surface", "clean neutral"
3. **Define composition:** "top-down view", "45-degree angle", "close-up", "wide angle"
4. **Set the style:** "commercial food photography", "documentary style", "minimalist product shot"
5. **Include quality markers:** "high resolution", "sharp focus", "shallow depth of field"
